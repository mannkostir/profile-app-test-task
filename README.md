# Profile App

## How To Run

1. Clone project:

   `git clone https://github.com/mannkostir/profile-app-test-task.git`

### Running client

1. Run terminal from the _root_ of the project
1. Go to **client** directory:

   `cd client`

1. Install dependencies:

   `yarn install`

1. Run client:

   `yarn run start`

### Running server

1. Run terminal from the _root_ of the project

1. Go to **server** directory:

   `cd server`

1. Install dependencies:

   `yarn install`

1. Run server:

   `yarn run start`
